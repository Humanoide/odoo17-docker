* situarse en :

/root/docker/portainer

(ahí estará el portainer y el docker-compose.yml con el que se creó portainer)


* haz copia de los datos :

tar -czf backup_portainer_data_$(date +%F).tgz portainer_data/

* después

docker compose pull
docker compose up -d
docker compose logs -f portainer
